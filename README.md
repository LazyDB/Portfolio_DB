# Dipesh Baral React Portfolio

A four-page React portfolio website using Vite, React Router, and a dark purple theme based on `#1a1a1a` and `#8936f5`.

## Pages

- Home
- Projects
- Resume
- Contact Me

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in your terminal.

## Resume PDF

The Resume page displays:

```text
public/assets/resume.pdf
```

To replace it later, keep the same filename or update the path in `src/pages/Resume.jsx`.
